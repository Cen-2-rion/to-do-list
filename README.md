# To-Do List

Простое приложение для управления задачами, созданное с использованием **React**, **TypeScript** и **Vite**.

---

## Запуск приложения

### 1. Клонирование репозитория

```bash
git clone git@github.com:Cen-2-rion/to-do-list.git
cd to-do-list
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск в режиме разработки

```bash
npm run dev
```

После запуска проект будет доступен по адресу: http://localhost:3000

---

### Структура приложения

    src/
      ├─ assets/         # Изображения (галочка, крестик)
      ├─ components/     # Компоненты приложения (Input, Item, List) и интерфейсы
      ├─ App.tsx         # Главный компонент приложения
      ├─ main.tsx        # Точка входа
      └─ App.css         # Общие стили

### Используемые технологии

* React — UI-библиотека

* TypeScript — типизация

* Vite — сборщик и dev-сервер

* CSS — стилизация

### Основной функционал

* Добавление задач

* Удаление задач

* Отметка выполненных (галочка)

* Сохранение состояния в памяти
